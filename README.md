# Shoe Release Discord Bot

This is a Discord bot that monitors Nike product releases and sends real-time updates to a Discord server. It works by scraping Nike’s website for upcoming or newly released sneakers and formatting that data into readable alerts.

## Getting Started

### Dependencies

* Python version 3.8 or higher
* Python packages:
```
python -m pip install discord.py
python -m pip install requests
python -m pip install pytz
```
* Discord developer tools

### Installing

* Get your Discord bot token from Discord Developer tools
* Clone the repo
```
git clone https://github.com/wstab/shoe-bot.git
```
* Enter your bot token in main.py
```
BOT_TOKEN = 'ENTER YOUR TOKEN'
```

### Executing program

* Run Program
  ```
  python main.py
  ```
